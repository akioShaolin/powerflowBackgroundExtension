# Como Instalar a Extensão PowerFlow Background Changer

## Pré-requisitos
- Google Chrome (versão 88 ou superior)
- Acesso ao site PowerFlow (https://powerflow.ecopower.com.br/)

## Passo a Passo da Instalação

### 1. Preparar os Arquivos
- Baixe todos os arquivos da extensão para uma pasta no seu computador
- Certifique-se de que todos os arquivos estão na mesma pasta:
  - `manifest.json`
  - `popup.html`
  - `popup.css`
  - `popup.js`
  - `content.js`
  - `icon16.png`
  - `icon48.png`
  - `icon128.png`

### 2. Abrir o Chrome
- Abra o navegador Google Chrome
- Digite na barra de endereços: `chrome://extensions/`
- Pressione Enter

### 3. Ativar o Modo Desenvolvedor
- No canto superior direito da página, você verá um botão "Modo do desenvolvedor"
- Clique para ativá-lo (o botão ficará azul)

### 4. Carregar a Extensão
- Clique no botão "Carregar sem compactação"
- Navegue até a pasta onde você salvou os arquivos da extensão
- Selecione a pasta e clique em "Selecionar pasta"

### 5. Verificar a Instalação
- A extensão aparecerá na lista de extensões instaladas
- Você verá o ícone da extensão na barra de ferramentas do Chrome
- Se não aparecer, clique no ícone de quebra-cabeça (extensões) e fixe a extensão

## Como Usar

### 1. Acessar o PowerFlow
- Navegue até https://powerflow.ecopower.com.br/
- Faça login normalmente

### 2. Abrir a Extensão
- Clique no ícone da extensão na barra de ferramentas
- Uma janela popup será aberta

### 3. Definir Plano de Fundo
- Cole a URL de uma imagem no campo "URL da Imagem"
- Clique em "Aplicar Fundo"
- A imagem será aplicada como plano de fundo

### 4. Remover Plano de Fundo
- Clique em "Remover Fundo" para voltar ao padrão

## Dicas Importantes

### URLs de Imagem Recomendadas
- Use imagens hospedadas em serviços confiáveis (Google Drive, Imgur, etc.)
- Certifique-se de que a URL termina com extensões como .jpg, .png, .gif
- Teste a URL em uma nova aba antes de usar na extensão

### Exemplos de URLs Válidas
```
https://images.unsplash.com/photo-exemplo.jpg
https://i.imgur.com/exemplo.png
https://drive.google.com/uc?id=SEU_ID_AQUI
```

### Resolução de Problemas

**A extensão não aparece:**
- Verifique se o modo desenvolvedor está ativado
- Recarregue a extensão na página chrome://extensions/

**A imagem não carrega:**
- Verifique se a URL está correta
- Teste a URL em uma nova aba do navegador
- Certifique-se de que a imagem é acessível publicamente

**O plano de fundo não aparece:**
- Verifique se você está no site correto (powerflow.ecopower.com.br)
- Recarregue a página do PowerFlow
- Tente aplicar a imagem novamente

## Desinstalação

Para remover a extensão:
1. Vá para `chrome://extensions/`
2. Encontre a extensão "PowerFlow Background Changer"
3. Clique em "Remover"
4. Confirme a remoção

## Suporte

Se encontrar problemas:
1. Verifique se todos os arquivos estão na pasta correta
2. Certifique-se de que o Chrome está atualizado
3. Tente recarregar a extensão
4. Reinicie o navegador se necessário
